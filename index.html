Lenguaje C# 2024

Este proyecto implica la creación de una aplicación web en C# utilizando ASP.NET Core con el objetivo de gestionar usuarios mediante operaciones CRUD (Crear, Leer, Actualizar, Eliminar). La aplicación sigue el patrón de diseño MVC (Modelo-Vista-Controlador) y emplea Entity Framework Core para la interacción con una base de datos SQL Server. Se desarrollan vistas para las acciones básicas de gestión de usuarios, tales como la visualización de la lista de usuarios, la creación de nuevos usuarios, la edición de detalles existentes y la eliminación de registros. Además, se implementa un manejo de mensajes de confirmación y errores para mejorar la experiencia del usuario.
